AgentFit – AI Powered Workout Generator
---------------
AgentFit is a full-stack web application that generates customized workout plans using artificial intelligence. The goal of the project is to simplify fitness planning by automatically creating workout routines tailored to the user's preferences and fitness goals.
Users can input details such as workout duration, fitness goal (e.g., muscle gain, fat loss, endurance), and experience level. The application sends this information to a backend server built with Node.js, which then interacts with the **Cohere API to generate a personalized workout routine. The generated workout is returned to the frontend and displayed dynamically for the user.
-----------
Key Features
AI-generated personalized workout plans
Real-time API integration with Cohere
Full-stack implementation using JavaScript
Dynamic frontend interaction using Fetch API
Simple and responsive user interface
